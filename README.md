# Apurva Raj — Personal Portfolio Website

A modern, ultra-performant personal portfolio website built with Next.js and Tailwind CSS. Live at [apurvaraj.pages.dev](https://apurvaraj.pages.dev).

## Tech Stack

- **Framework:** Next.js 16 (App Router, Static Site Generation)
- **Styling:** Tailwind CSS v4
- **Deployment:** Cloudflare Pages (Free Tier)
- **Icons:** Lucide React

## Features

- Typewriter effect on hero section
- Auto-rotating project carousel with pause on hover
- Vertical timeline for education section
- Mobile responsive with hamburger menu
- Optimized for all screen sizes from mobile to 4K Smart TV
- Custom cyan scrollbar
- Smooth scroll navigation
- SEO optimized with metadata, robots.txt and sitemap.xml
- Static site generation for maximum performance

## Local Development

Clone the repository:
```bash
git clone https://github.com/apurvaraj9/apurva-portfolio.git
```

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Every push to the `main` branch automatically triggers a redeployment on Cloudflare Pages.

To build locally:
```bash
npm run build
```
